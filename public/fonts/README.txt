Place the following OpenDyslexic font files in this directory:

  - OpenDyslexic-Regular.woff2
  - OpenDyslexic-Bold.woff2

OpenDyslexic is an open-source font licensed under the SIL Open Font Licence.
Download it from: https://opendyslexic.org

If these files are absent, the Dyslexia Friendly feature will fall back to
Comic Sans MS then Arial. The feature remains functional; only the specific
font changes.
